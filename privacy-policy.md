# Política de Privacidade - Controle Financeiro

**Última atualização**: Janeiro de 2026

## 1. Informações Coletadas

### 1.1. Informações de Conta

- Email e nome para autenticação
- Foto de perfil (opcional)

### 1.2. Dados Financeiros

- Transações (receitas, despesas, investimentos, ofertas)
- Metas financeiras
- Orçamentos mensais
- Histórico de dízimos e ofertas

### 1.3. Dados de Uso

- Logs de erros (anônimos)
- Estatísticas de uso (anônimas)

## 2. Como Usamos Seus Dados

- Fornecer funcionalidades do aplicativo
- Sincronizar dados entre dispositivos
- Realizar backup em nuvem
- Melhorar o serviço

## 3. Armazenamento e Segurança

- Dados armazenados no Firebase (Google Cloud Platform)
- Criptografia em trânsito (HTTPS)
- Criptografia em repouso
- Acesso protegido por autenticação

## 4. Compartilhamento de Dados

**NÃO compartilhamos seus dados financeiros pessoais com terceiros.**

Compartilhamento limitado:

- Firebase/Google (infraestrutura)
- Apenas dados necessários para operação

## 5. Seus Direitos

Você tem direito a:

- Acessar seus dados
- Exportar todos os dados
- Deletar sua conta
- Revogar permissões

## 6. Retenção de Dados

- Dados mantidos enquanto conta ativa
- Após exclusão: dados deletados em até 30 dias
