# 💰 Controle Financeiro App

App de controle financeiro pessoal com funcionalidades específicas para gestão de dízimos e ofertas.

## 🚀 Funcionalidades

- ✅ Gestão de Receitas e Despesas
- ✅ Controle de Investimentos
- ✅ Calculadora de Dízimo (10% automático)
- ✅ Registro de Ofertas
- ✅ Metas Financeiras
- ✅ Planejamento Mensal
- ✅ Relatórios Avançados
- ✅ Backup na Nuvem
- ✅ Sistema Premium

## 📱 Tecnologias

- React Native
- Firebase (Auth + Firestore)
- React Navigation
- Zustand (State Management)
- Gifted Charts (Gráficos)

## 🛠️ Instalação
```bash
# Instalar dependências
npm install

# Rodar no Android
npx react-native run-android

# Rodar no iOS
npx react-native run-ios
```

## 📂 Estrutura do Projeto
```
src/
├── assets/          # Imagens, ícones, fontes
├── components/      # Componentes reutilizáveis
├── navigation/      # Configuração de rotas
├── screens/         # Telas do app
├── services/        # Firebase, APIs, Storage
├── store/           # Gerenciamento de estado
└── utils/           # Helpers e constantes
```
